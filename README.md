## 🚀 Featured Projects

### 📚 [LAB LIB](https://github.com/parchurraja/LAB-LIB-)
> *An Intelligent Multi-Category Book Discovery & Cataloging Platform.*
- **Tech Stack:** Spring Boot, Spring Security, JWT, MySQL, Jsoup, React, Framer Motion.
- **Key Features:**
  - Automated Jsoup scraping engine traversing target catalog structures recursively.
  - Stateful user wishlist, dynamic shopping cart persistency, and custom search history debounce filters.
  - Scheduled data sync jobs via Spring Scheduler.

### 📰 [Global News Aggregator](https://github.com/parchurraja/Global-News-Aggregrator)
> *A Real-time Intelligent News Gathering & Curation Platform.*
- **Tech Stack:** Spring Boot, Spring Security, JWT, MySQL, HTML5/CSS3/JavaScript.
- **Key Features:**
  - Ingests and aggregates real-time news articles from various RSS/XML feeds (Google News India, TechCrunch, RBI).
  - Secure stateless JWT session handling, rate-limiting filter, and safe user account management.
  - Built-in deduplication pipeline ensuring fresh and high-quality feed output.
